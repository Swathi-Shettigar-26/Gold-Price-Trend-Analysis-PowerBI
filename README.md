🪙 Gold Price Trend and Market Analysis using Power BI

📌 Project Description

This project performs exploratory and time series analysis on historical gold price data to study market behavior, volatility patterns, and trend movements. An interactive Power BI dashboard was developed to transform raw financial data into meaningful analytical insights using data modeling, DAX measures, and forecasting techniques.

📊 Dataset Information

The dataset contains historical gold market data with the following fields:

Date

Open

High

Low

Close

Volume

The data represents daily gold price movements and trading activity.

🎯 Project Objectives

Perform trend analysis on gold closing prices

Identify peak and lowest price levels

Analyze price volatility using high low spread

Examine trading volume behavior

Implement time series forecasting

Conduct scenario analysis using What If parameters

🛠 Tech Stack

Microsoft Power BI

Power Query Editor for ETL

DAX for calculated columns and measures

Built in Power BI Forecasting

Data Modeling and Time Intelligence

⚙️ Methodology

Data Cleaning

Verified data types

Removed inconsistencies

Structured date hierarchy

Data Modeling

Created Date table for time intelligence

Developed calculated measures such as:

Average Close Price

Highest and Lowest Price

Total Volume

Price Movement Indicator

Visualization

Line charts for trend analysis

Donut charts for price movement distribution

Bar charts for volume comparison

KPI cards for quick insights

Slicers for dynamic filtering

Forecasting

Applied time series forecasting on closing price

Configured confidence interval

Generated short term future predictions

Scenario Simulation

Implemented What If parameter

Simulated growth percentage impact on price

📈 Key Insights

Identified long term upward and downward market trends

Detected high volatility periods using price spread

Observed correlation between volume spikes and price changes

Generated predictive insights for future price direction

🚀 Deployment

The report was published to Power BI Service for interactive access and sharing. Scheduled refresh can be configured to ensure up to date analysis.
